# Orthoph-on

## Require
* NodeJS
* NPM
* Electron

## Run
Make sure you're in the right folder with your terminal then:
1. Run ``npm install`` if it has not been done before
2. Then run ``npm start``

## Build
### For Windows
```
npm run package-win
```
### For MacOS
```
npm run package-mac
```
### For Linux
```
npm run package-linux
```
